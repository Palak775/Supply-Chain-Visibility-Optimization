# Supply Chain Analytics - Milestone 1

## Project Title
SupplyChain_Visibility_Optimization

## Objective
1. Import supply chain dataset into Power BI
2. Perform data preprocessing - handle missing values, correct data types
3. Design a star schema data model with fact and dimension tables

## Dataset Source
DataCo Supply Chain Dataset

## Data Cleaning and Transformation Steps
- Handled missing values and duplicates
- Corrected data types
- Renamed columns for clarity
- Applied transformations using Power Query

## Data Model Overview
Designed Star Schema with Fact and Dimension tables. 
Defined Primary and Foreign key relationships.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
